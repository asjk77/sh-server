# sh_server

Express 기반 ServerModule입니다.


##  dependencies 모듈 설치
의존 모듈을 설치하기 위해서는 다음 스크립트를 실행하세요
```
npm install
```

## run
서버를 실행하기 위해서는 다음 명령어를 입력하세요.
```
npm start
```

## test
`npm test` 명령어로 테스트를 수행할 수 있습니다.
`test/api/` 디렉토리로 rest test를 할 수 있습니다.

## Project Config
프로젝트 최상위 폴더 안에 ./env 파일이 필요합니다.
.env 파일을 설정하세요!
